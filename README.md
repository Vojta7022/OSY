# OSY – CTU Operating Systems

This repository contains my coursework, homework solutions, and systems programming materials for the CTU Operating Systems course.

## About

I keep this repository public mainly as a personal archive of my work and as a study resource for other students.  
It includes homework folders, scripts, and supporting materials created while working through operating systems topics during the course.

## Contents

This repository may include:
- homework assignments
- system-level programming tasks
- shell scripts
- Makefile-based builds
- memory and process related exercises
- supporting materials for course work and testing

## Main Topics

The materials in this repository relate to topics such as:
- processes and threads
- memory management
- page faults and memory maps
- low-level systems programming
- shell scripting
- build tooling and debugging

## Structure

The repository is organized mainly into homework and task folders, including:
- `hw3` to `hw6`
- `NOVA_HW10` to `NOVA_HW12`
- additional helper scripts and supporting files

Some folder names reflect the original course workflow and assignment structure.

## Notes

This is primarily a coursework and study repository, not a polished standalone software project.  
Some files are kept in their original working structure from the course.
